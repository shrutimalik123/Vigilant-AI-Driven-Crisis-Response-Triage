# Vigilant - Project Roadmap

## Phase 1: Core Triage (Current)
- [x] Real-time Responder Dashboard with Map integration.
- [x] Voice-enabled User PWA (Simulated).
- [x] AI Triage Logic (Keywords to Severity).
- [x] Socket.io Real-time Alerts.

## Phase 2: Enhanced AI & Integration (Next Steps)
- [ ] **ElevenLabs Integration**: Replace simulation with actual API calls for Low-Latency Speech-to-Speech.
- [ ] **Nexos.ai Orchestration**: Implement robust state machine for complex conversation flows.
- [ ] **Wearable Integration (IoT)**: 
    - Connect to Apple Watch / Fitbit APIs.
    - Real-time heart rate monitoring during emergency calls.
    - Automatic stress level detection based on HRV (Heart Rate Variability).
- [ ] **Multi-Language Support**: Enable real-time translation for non-English speakers.

## Phase 3: Smart City Deployment
- [ ] **Municipal API Hooks**: Integrate with existing 911 dispatch systems (CAD).
- [ ] **Drone Dispatch**: Auto-deploy drones to GPS pins for visual confirmation.
- [ ] **Offline Mode**: Local PWA capabilities for disconnected scenarios.
